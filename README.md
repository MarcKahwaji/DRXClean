# DRX Premium Transportation

Official website for DRX Premium Transportation - Syria's trusted transportation partner since 1993.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

Build the project with `npm run build`, then deploy the `dist` folder to any static hosting provider.
